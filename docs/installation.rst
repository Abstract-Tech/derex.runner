============
Installation
============


Stable release
--------------

To install derex.runner, run this command in your terminal:

.. code-block:: console

    $ pip install derex.runner

This is the preferred method to install derex.runner, as it
will always install the most recent stable release.

If you don't have `pip <https://pip.pypa.io>`_ installed, this
`Python installation guide <https://docs.python-guide.org/starting/installation/>`_
can guide you through the process.

From sources
------------

The sources for derex.runner can be downloaded from the
`Github repo <https://github.com/silviot/derex.runner>`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/silviot/derex.runner

Or download the `tarball <https://github.com/silviot/derex.runner/tarball/master>`_:

.. code-block:: console

    $ curl  -OL https://github.com/silviot/derex.runner/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install
